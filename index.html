Module Grades Visualiser

Click the image to open the app or click here to view on github

An app built with Python and Streamlit, used to visualise the average grades at the University of St Andrews. Users have the option to view summary statistics or filter data depending on their required module, department or level. The graphs displaying the data are plotted using Plotly. Users can also search for a specific module by using the search bar.

Vector Drawing Package

Click here to view on github

A drawing application that I built using Java Swing. Users can draw draw various different shapes onto the canvas in different colours, with or without fill. Images can also be imported onto the canvas and are scaled based on the magnitude of the mouse drag. Amendments can be easily made to drawings through the undo/redo buttons. Previous drawings can be loaded onto the canvas and new drawings can be saved.

House price calculator
