Meta Internship

• Worked with the Central Notifications Platform team, which builds the frameworks and services that send notifications for Meta's family of apps (e.g., Facebook, Instagram, and WhatsApp).

• Stopped ~1 billion unnecessary notifications from being logged to our tables daily by identifying and removing a Facebook notification that was not being sent to anyone but was still being generated.

• Modified loggers to increase the probability that the content ID of a notification is logged. The changes reduced the proportion of null content IDs logged by more than 80%.

• Introduced functionality in two internal tools used across Meta, which allow developers to control the logging rate of their notification.

• Designed and implemented a smart sampling configuration that logs notifications at a rate decided based on the notification’s level of importance.

Arm Internship

• Worked with the CPU team to model the behaviour of various CPU counters such as cache access & clock cycles.

• Implemented a data processing pipeline in Python which automatically validates a dump of counter information from each core and generates further CSV files based on various conditions; therefore helping engineers spot issues.

• The tool is saving over 30 minutes worth of time for each test run (as opposed to manual analysis) and has helped in spotting an issue with the way the CPUs are released before execution of test code.

• Built a classifier to predict the final state of a CPU based on counter information; this achieved an F-Score of 0.98.

Fall Finder

Click here to read more

Worked with a team in Arm’s Global Hackathon to implement a Fall Detection System, with the intention of it being used in a hospital/care home setting, as it allows for a faster response from carers. The system uses Python along with OpenPose, OpenCV and TensorFlow to detect key points and estimate the pose of the human body using pre-trained deep learning models. The model can learn from a database of images of people and can use this information to classify a patient. The system detects when a person falls, and an email is sent to a carer which includes an attached picture of the fall and the person’s name.

Optiver Trading Competition

Wrote algorithms in Python in order to execute trades on a simulated trading exchange, with the main aim being to make a profit. I Implemented the logic needed to check the market for potential arbitrage opportunities constantly, if the prices of an asset diverged across markets by a given threshold, the program would execute a trade. Made us of a Relative Strength Index (RSI) implementation to capture trends in market prices and leveraged hedging to offset risks, therefore protecting against large swings in the market. As a result, my team was one of the few teams that generated a profit of over 1 million USD in virtual currency.
